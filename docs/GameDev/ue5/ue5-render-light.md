# Light Types
灯光类型：
1. Directional Loght 平行光
2. Point Light 点光源
3. Spot Light 聚光灯
4. Rect Light 区域光
5. Sky Light 天光
捕捉远处静态光线，e.g.天空球,作用类似于光照烘焙。

## light bleed
光照出血：一些老的材质不支持流明而出现渲染错误或异常。需要修改材质编辑器以解决问题。
